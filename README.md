# Alta Gracia Missions — Bringing Hope to the World

Alta Gracia Missions is a 501(c)(3) nonprofit organization serving communities through faith, outreach, and compassionate action. From our roots in the Philippines to mission fields across the globe, we've been walking alongside communities for over two decades, offering practical help and lasting spiritual impact.

## Features
- **Responsive Design**: Mobile-first approach for all devices.
- **Dynamic Interactions**: Smooth scroll, animations, and interactive elements.
- **Secure Donations**: Integration with Zeffy for secure, tax-deductible giving.
- **Multimedia Content**: Embedded YouTube mission videos and image galleries.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Hosted on Microsoft Azure
